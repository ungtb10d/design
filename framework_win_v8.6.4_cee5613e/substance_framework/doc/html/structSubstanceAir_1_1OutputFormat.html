Detailed Description

Structure used for output format/layout override Allows to override pixel format, mipmap pyramid depth, horizontal and vertical flipping, resize, per component value remapping (e.g. invert), components shuffling and composition from other graph outputs (e.g. allows channels composition to match shader layout).

unsigned int SubstanceAir::OutputFormat::mipmapLevelsCount

Mipmap pyramid depth

  • UseDefault: Default original mipmap pyramid depth (default value).
  • MipmapFullPyramid: full mipmap pyramid.
  • MipmapNone: Only one level, no mipmaps.
