Detailed Description

Substance render options Used at renderer creation and by Renderer::setOptions method. Allows to set memory consumption budget and CPU usage.

◆ mMaxIntermRenderSize

size_t SubstanceAir::RenderOptions::mMaxIntermRenderSize

Maximum size (log2) of internal renders This can be used to limit the size of the intermediate nodes computed by the engine, so that nodes whose width/height are higher than the threshold are computed at the threshold instead.

Note
This limit is in log2 format, i.e. setting it to 12 limits the width/height to 2048. Default is 15 (32768).
