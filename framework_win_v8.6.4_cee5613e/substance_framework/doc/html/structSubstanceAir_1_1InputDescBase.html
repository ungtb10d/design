◆ mAlteredOutputUids

Uids SubstanceAir::InputDescBase::mAlteredOutputUids

List of output UIDs altered by this input UIDs are sorted (allows binary search).

When this input is modified, the outputs listed here are flagged as dirty (outdated) and are recomputed at next rendering.

Warning
Dynamic output components composition (OutputFormat::Component::outputUid) is not taken into account.
